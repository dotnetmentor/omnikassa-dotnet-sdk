Detailed Description

The token provider is needed to store and retrieve the values of a token in key value pairs. The abstract class requires that the GetValue(TokenProvider.FieldName) and the SetValue(TokenProvider.FieldName, string) functions are implemented. The implementation is entirely up to the developer. For example the values can be persisted in a database and cached. The only thing to note, however, is that the refresh token is provided by your implementation and should have been supplied by the Rabobank together with this SDK release.
